## 🚀 Deployment Instructions

### Option 1: Deploy to GitHub Pages

1. **Create a GitHub repository** (if you don't have one):
   ```bash
   git init
   git add .
   git commit -m "Initial portfolio commit"
   git branch -M main
   git remote add origin https://github.com/yourusername/portfolio-website.git
   git push -u origin main
   ```

2. **Enable GitHub Pages**:
   - Go to your repository settings
   - Scroll to "GitHub Pages" section
   - Select "Deploy from a branch"
   - Select "main" branch, root folder
   - Click Save

3. **Access your site**:
   - Your portfolio will be available at: `https://yourusername.github.io/portfolio-website/`

### Option 2: Deploy to Netlify

1. **Drag and drop**:
   - Visit [netlify.com](https://netlify.com)
   - Sign up with GitHub
   - Drag and drop the portfolio folder
   - Your site goes live instantly

2. **Or connect your repository**:
   - Connect your GitHub account
   - Select the portfolio repository
   - Set build command: (leave empty)
   - Set publish directory: `.`
   - Click Deploy

### Option 3: Deploy to Vercel

1. **Connect repository**:
   - Visit [vercel.com](https://vercel.com)
   - Import your GitHub repository
   - Click Deploy
   - Your site is live!

## 📝 Customization

### Update Personal Information
Edit the content in HTML files to reflect your information:
- Change name from "Noah Thérowa" to your name
- Update bio and technical skills
- Modify contact email and links
- Add your social media profiles

### Customize Colors
Edit CSS variables in `css/style.css`:
```css
:root {
    --primary-color: #1a73e8;        /* Main color */
    --primary-dark: #1565c0;         /* Hover state */
    --secondary-color: #34a853;      /* Secondary */
    --accent-color: #fbbc04;         /* Accent */
    /* ... more variables */
}
```

### Add Your Projects
Edit `projects.html` to add/remove projects. Follow the same structure:
```html
<article class="project-card">
    <div class="project-header">
        <h2>Your Project Title</h2>
        <span class="project-type">Project Type</span>
    </div>
    <!-- Add project details -->
</article>
```
