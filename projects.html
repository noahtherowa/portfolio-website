Rower Speed Predictor

ML / Sports Analytics

Problem Statement

A machine learning model trained on real rowers' ages and 2K erg times to predict a user's expected rowing speed. Gives athletes a data-driven benchmark based on where they fall in the broader rower population.

Technical Stack

  • Language: Python
  • Libraries: Scikit-learn, Pandas, NumPy
  • Techniques: Regression modeling, Feature engineering, Model evaluation
  • Data: Rower age and 2K split time dataset

Process & Reflection

Collected and cleaned real-world rowing performance data, then trained a regression model to surface meaningful speed predictions from just two inputs: age and 2K time. The challenge was working with a relatively small, noisy dataset while still producing reliable predictions.
